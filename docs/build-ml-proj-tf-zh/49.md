# Inception v3

快进到 2015 年 12 月，Inception 架构有了新的迭代。 两次发行之间月份的不同使我们对新迭代的开发速度有了一个想法。

此架构的基本修改如下：

*   将卷积数减少到最大 3x3
*   增加网络的总体深度
*   在每一层使用宽度增加技术来改善特征组合
