# 最近邻方法

本章将重点介绍最近邻方法，以及如何在 TensorFlow 中实现它们。我们将首先介绍这些方法，然后我们将说明如何实现各种形式。本章将以地址匹配和图像识别的示例结束。

在本章中，我们将介绍以下内容：

*   与最近邻居合作
*   使用基于文本的距离
*   计算混合距离函数
*   使用地址匹配示例
*   使用最近邻居进行图像识别
