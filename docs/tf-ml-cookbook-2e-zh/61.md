# 卷积神经网络

卷积神经网络（CNN）负责过去几年中图像识别的重大突破。在本章中，我们将介绍以下主题：

*   实现简单的 CNN
*   实现先进的 CNN
*   重新训练现有的 CNN 模型
*   应用 Stylenet 和神经式项目
*   实现 DeepDream
